# Impostor

Mini-juegos para jugar entre amigos, sin instalar nada. Cada juego es un archivo HTML autocontenido: lo abrís en el navegador del celu y listo.

## Juegos

- **`index.html` — El Impostor.** El juego de descubrir al impostor.
- **`pictionary.html` — Dibujá, Che.** Pictionary mobile bien argento (asado, fernet, modismos, fútbol y más). Modo "pasá el celu", canvas para dibujar, cronómetro, equipos y tablero de puntajes.

## Dibujá, Che (Pictionary)

App mobile-first (React + Canvas), sin backend y sin registro: todo el estado vive en memoria.

Cómo jugar:
1. Abrí `pictionary.html` en el celu.
2. Armá 2+ equipos y sumá jugadores.
3. Elegí categorías, dificultad, tiempo y rondas (y si va, el modo after 🌶️).
4. Pasate el celu: el dibujante mira la palabra y dibuja mientras su equipo adivina.

Incluye: lápiz con grosor ajustable, goma, deshacer y limpiar, paleta de colores, cronómetro con tic-tac/fanfarria/bocinazo, vibración, muerte súbita para desempate y estadísticas finales.
